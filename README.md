MoodLight
==============
 Semester mini project for simple IoT. It's just a mood light with dry flower in glass bottle. Mood light can be controlled by using internet.

Environment
-----------

* Board  
  * Raspberry Pi  
* OS  
  * Raspbian  
* Framework  
  * Node.JS  
* Libraries  
  * wiring-pi  
  * node-speaker  
  * express  
  * fs  
* WebPage  
  * Bootstrap
